1. Open Ethereum and Folder Projects
2. Copy the file (CovidMedicalRecords.sol) into Ethereum IDE
3. Open Terminal in Vscode Projects
4. Activate the Hardhat Local
	"npx hardhat node"
5. Import Account (use private account) into Metamask
6. Set the Custom Network 
| ------------------ | ---------------------------------------------- |
| Network Name       | Hardhat Localhost                              |
| RPC URL            | [http://127.0.0.1:8545](http://127.0.0.1:8545) |
| Chain ID           | 31337                                          |
| Currency Symbol    | ETH                                            |
| Block Explorer URL | (kosong)                                       |
7. Deploy Contract in Ethereum (using Injected Metamask)
8. Insert the new Contract Address and Contract ABI into all file.js
9. Run the react app (cd into destinated folder where you save the react app)
	"npx run start"
